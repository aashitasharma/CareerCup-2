 * Find a duplicates in an array of length n. 
 * The values are positive integers in the range between 1 .. n-1
 */

Solution:
---------
Implemented without hashmap.
you have to preserve original array.
Negate the numbers to preserve the value but mark as visited.
Time Complexity = O(1) 
Space Complexity = 0 (no additional space required.)
